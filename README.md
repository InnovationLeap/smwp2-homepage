# SMWP 2 官网

SMWP 2（Super Mario Worker Project 2）是由 INNOVATION LEAP 开发的全新 Mario Worker，在尽可能兼容 SMWP 1 关卡体验的基础上，为 SMWP 带来更多的可扩展性。

SMWP 2 官网使用 [bun](https://bun.com/) + [VitePress](https://vitepress.dev/) 构建。

## 快速开始

### 安装依赖
```bash
bun install
```

### 开发环境
```bash
bun run docs:dev
```

### 构建
```bash
bun run docs:build
```
